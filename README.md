# Skate

Miniramp warmup trick checklist. Track your transition skating progress with localStorage persistence.

Hosted at [skate.strangelabs.io](https://skate.strangelabs.io)

## Structure

```
skate/
├── index.html      # App entry point
├── css/
│   └── style.css   # Styles (extracted from Strange Labs hub)
└── CNAME           # GitHub Pages custom domain
```

## Development

Open `index.html` in a browser. No build step required.

## Deployment

Served via GitHub Pages with a custom domain. Push to `main` to deploy.
